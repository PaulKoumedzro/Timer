# Digital Clock for your computer.

-The Digital clock was design as a web aplication to display time in the browser while you work.Also enhance your day with a daily motivational quotes.I used HTML/CSS/Javascript to build the website.
-10% of CSS.
-20% of HTML.
-70% of Javascript.
 -The hardest part was how to have the AM change to PM when it's 12PM for example.I have a plan to implement an APi in the future to have a massive quote database instead of storing them in an array.
 
 # How to Use the Project?
 -To use the project, you must first download the zip to your computer,extract it and open it in your text editor.You also need to buy a domain name if you choose to host and launch it.
